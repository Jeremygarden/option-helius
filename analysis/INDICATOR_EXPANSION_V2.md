# Macro Indicator Expansion v2

## Sigmoid评分函数设计原则

### 为什么用Sigmoid？
1. **无悬崖效应**：避免了在硬阈值附近（如 74.9 vs 75.1）产生的跳变。
2. **参数有物理意义**：`center` 代表金融逻辑上的"中性"值。
3. **抗过拟合**：参数基于长期金融逻辑设计，而非对历史噪音的简单拟合。

### center参数如何确定？
`center` = 历史"中性"值，而非"危险触发"值。
- 例如 **CAPE**: `center=25`（现代平均水平），而不是 `center=30`（危险阈值）。
- 例如 **VIX**: `center=18`（长期平均水平）。

### 动量参数 α=0.15 的含义
最多放大/缩小 ±30%（`1.0 + 0.15 * z_clamped`，z capped at ±2.0）。
- 如果某指标在 3 个月内快速向危险区移动（如 VIX 骤降或 PE 骤升），得分将上浮，捕捉"恶化动量"。
- 防止在高位盘整时低估风险，也能在风险快速改善时降低虚报。
