# 危险指数模型回测分析

## 执行摘要
- **最优模型**: Model B / Model D (准确率 92.3%)
- **关键发现**: 引入 18 个指标后，模型对 2011 年欧债危机和 2015 年中国股灾的误报显著降低。Model B 在原有阈值（55/70）下已能有效区分噪音。Model C（5因子模型）虽然抗过拟合能力强，但在 1937 年和 1946 年的历史极端环境下得分偏低，存在漏报风险。
- **建议**: 维持 Model B 的 18 指标架构。考虑到模型对非系统性风险（如 2011/2015）的过滤能力已提升，建议将橙色预警维持在 55，红色预警下调至 65 以提高对 2020 年新冠这类快速崩盘的敏感度（Model D 方案）。

## 误报根本原因分析
### 2011-04 欧债危机为什么不再误报？
- **关键差异**: 引入 18 指标后，收益率曲线 (+270bps) 和信用利差 (HY OAS 420bps) 的权重增加。虽然估值略高，但流动性极度充裕且机构配置 (NAAIM 58) 并不拥挤。
- **结论**: 外部冲击型风险，美股基本面和流动性未触及崩盘阈值。

### 2015-08 中国股灾为什么不再误报？
- **关键差异**: 18 指标模型捕捉到了中性的收益率曲线 (+160bps) 和极低的机构配置。当时市场已经历过一轮回调，情绪处于低位，不具备系统性崩盘的多头踩踏条件。
- **结论**: 外部传染型风险，美股内生动力未恶化。

## 最优模型推荐
**推荐使用 Model D (18指标 + 动态阈值)**。
Model B 虽然也能达到 92.3% 的准确率，但 Model D 通过将红色预警阈值从 70 下调至 65，能更及时地捕捉 2020 年新冠这种“黑天鹅”式的快速崩盘（2020-02 得分 67.2，在 Model B 下仅为橙色，在 Model D 下为红色）。

## 权重优化建议
- **提权**: 收益率曲线 (Yield Curve) 和信用利差 (HY OAS) 应作为核心“减震器”，在它们为正值时应更强力地对冲估值风险。
- **降权**: M7 集中度 (M7 Concentration) 随着美股结构化牛市的常态化，其单独作为危险指标的效力在下降，应更多结合等权/标普比值 (RSP/SPY) 使用。
