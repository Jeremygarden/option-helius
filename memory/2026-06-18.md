## 技术架构备忘

### Ralph 调用规范（当前已验证可用）
```
主session → sessions_spawn(subagent) → exec: ralph --prompt-file <file> --max-iterations 1 --completion-promise COMPLETE --no-commit
  └─ openclaw-agent → openclaw agent --message <prompt>
       └─ financer agent 做代码修改
```
- 每轮 ralph 后必须手动 git commit + push
- `rm -f .ralph/ralph-loop.state.json` 避免 state 残留
- `git pull --rebase origin main` 在每次 push 前执行
- 无 TTY 依赖，subagent 内 exec 即可运行
